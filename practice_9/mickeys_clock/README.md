# Mickey Clock Application

## Description
This project is a simple clock created using Pygame.
It displays the current system time using Mickey Mouse hands.

The same hand image is used twice:
- One for minutes
- One for seconds

The clock updates in real time.

---

## Features
- Displays current system time (minutes and seconds)
- Uses rotation for clock hands
- Updates every second
- Simple graphical clock interface

---

## Controls
- Close window to exit the program

---

## Technologies
- Python
- Pygame
