# TSIS 3 Racer Game

Advanced Pygame Racer with lane hazards, dynamic road events, traffic cars, power-ups, leaderboard, and settings.

## How to run

```bash
python main.py
```

## Features

- Main menu: Play, Leaderboard, Settings, Quit
- Username input before starting the game
- Dynamic traffic cars
- Road obstacles: barriers, oil/potholes
- Safe spawn logic
- Power-ups: Nitro, Shield, Repair
- Difficulty scaling
- Score, distance, coins, and lives display
- Persistent leaderboard saved to `leaderboard.json`
- Settings saved to `settings.json`
- Settings screen with sound toggle, car color, and difficulty

## Controls

- Arrow keys: move car
- R: retry after game over
- M: return to main menu after game over
